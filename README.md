## 📖 English

### What is blackPad?

**blackPad** is a clean, distraction-free note-taking app built with Flutter. Designed around a pure black theme, it keeps your focus where it belongs — on your words.

### ✨ Features

- 🌑 **Dark Theme** — A true black UI, easy on the eyes in any lighting
- 🔍 **Search** — Instantly find any note by keyword
- 📌 **Pin Notes** — Keep your most important notes always at the top
- 📁 **Folder Organization** — Organize notes into custom folders (e.g. Developer, Personal)
- 📴 **Offline Support** — All your notes are stored locally, no internet needed
- 🗑️ **Recently Deleted** — Recover accidentally deleted notes

### 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/blackPad.git

# Navigate to project directory
cd blackPad

# Install dependencies
flutter pub get

# Run the app
flutter run
```
